Default IV updates

The teambuilder now minimizes Attack IVs on Pokémon that don't run physical moves (while maintaining the type of the move Hidden Power, if present). This has been the default on Smogon's recommended movesets for a while, as it reduces the damage taken from Foul Play and confusion. Trick Room and Gyro Ball behave similarly, but minimize Speed IVs instead.
